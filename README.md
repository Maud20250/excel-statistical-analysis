# Excel Statistical Analysis Projects

## Overview
Two statistical analysis projects demonstrating advanced Excel and 
data analysis skills including probability, regression, hypothesis 
testing, and time series analysis.

## Project 1 - Customer Purchase Analysis
- Dataset: 60 customers with age, gender, city, purchase history
- Analysis: Mean, Median, Mode, Goal Seek
- Categories: Electronics, Clothing, Home Decor

## Project 2 - Diamonds and Star Sales Analysis
- Part A: Probability and frequency distribution
- Part B: Descriptive statistics, variance, correlation
- Part C: Multiple linear regression (Y = β₀ + β₁X₁ + β₂X₂)
- Part D: Binomial, Poisson, Normal distributions
- Part E: Hypothesis testing (t-test)
- Part F: Time series analysis (least squares method)

## Tools & Technologies
- Microsoft Excel
- Statistical Analysis
- Data Visualization
